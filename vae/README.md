# Variational Autoencoders (VAEs) in PyTorch

This module contains from-scratch PyTorch implementations of Variational Autoencoders (VAEs).

## Implementations

- Basic VAE on MNIST
  - Reparameterization trick
  - ELBO (evidence lower bound) objective
  - Sampling and reconstruction visualizations

Planned extensions:

- Convolutional VAE on image datasets (e.g., CelebA)
- Î²-VAE and disentanglement experiments
