# RealNVP (PyTorch From Scratch)

This folder contains a full from-scratch PyTorch implementation of the **Real Non-Volume Preserving (RealNVP)** normalizing flow model.

## Contents

- `realnvp_from_scratch.ipynb`  
  End-to-end implementation including:
  - Coupling layers
  - Forward & inverse transformations
  - Log-determinant Jacobian
  - Training on toy datasets
  - Sampling & visualization

This implementation is designed for:
- Learning normalizing flows deeply
- Educational use
- Research prototyping
